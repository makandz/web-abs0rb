# web-abs0rb

Archive site for **Abs0rb.me**, published after the original game was taken offline.

Abs0rb.me is now offline, this project exists to preserve a nostalgic snapshot of the game with historical stats, leaderboards, and fun memories pulled from the old database and exported as static JSON.

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm / yarn / pnpm

### Install

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

### Build

```bash
npm run build
```

### Start (production)

```bash
npm run start
```
